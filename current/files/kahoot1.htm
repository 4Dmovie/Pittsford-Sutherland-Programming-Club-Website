15.          What's the greatest common divisor (gcd) of 12549 and 7379.

A.  1 B. 47 C. 141 D. 987

By using Euclid's theorem, (12549, 7379) = (7379, 5170) = (5170, 2209) = (2209, 752) = (752, 705) = (705, 47) = (47, 0), so gcd(12549, 7379) = 47.

Or we can factor the two numbers:

12549 = 3 * 47 * 89,

7379 = 47 * 157,

The largest common factor is 47.
